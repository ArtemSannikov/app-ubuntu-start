badblocks

На главную

Описание

Команда badblocks выполняет проверку раздела на наличие bad-блоков.

Синтаксис: [Команда] [Параметры] [Раздел]

# Будет выполнена проверка раздела hda1 на наличие bad-блоков

$ badblocks -v /dev/hda1

Параметры

-n - заставляет использовать режим чтения-записи, который не разрушает данные. Этот режим используется по умолчанию для выполнения проверки диска.

-s - отображает ход процесса проверки, выводя номера блоков в том порядке, как они проверялись.

-v - подробно сообщать о происходящем.

-w - для проверки используется режим реальной записи. С этим параметром команда badblocks выполняет поиск сбойных блоков, сначала записывая специальные образцы (0xaa, 0x55, 0xff, 0x00) в каждый проверяемый блок устройства, затем читая каждый блок и сравнивая его содержимое делает вывод.
