Кнопка назад

head

На главную

Описание

Команда head позволяет просмотреть начало файла. По умолчанию предполагается вывод только первых десяти строк. Количество выводимых строк можно изменить, указав соответствующий ключ.

Синтаксис: [команда] [-n] [файл]

#Вывод 15 первых строк файла test.txt

$ head -15 test.txt

Параметры

-n - ключ, который отвечает за количество выводимых строк.
