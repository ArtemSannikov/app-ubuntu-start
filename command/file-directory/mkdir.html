mkdir

На главную

Создание директории

Команда (mkdir) позволяет создать новую директорию в терминале операционной системы Ubuntu.

$ mkdir [dir1]

Создание одновременно двух директорий

Команда (mkdir) позволяет создать одновременно две директории в терминале операционной системы Ubuntu.

$ mkdir [dir1] [dir2]
