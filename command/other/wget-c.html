wget -c [example.ru]

На главную

Загрузка файла с сайта

Команда wget -c позволяет загрузить любой файл с сайта, с возможностью остановки и продолжения загрузки.

$ wget -c [example.ru] //команда
$ wget -c www.artemsannikov.ru/test.iso //система будет загружать файл test.iso с сайта www.artemsannikov.ru
