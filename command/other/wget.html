wget

На главную

Описание

Утилита wget предназначена для загрузки файлов из интернет. Она поддерживает протоколы HTTP, HTTPS, и FTP, загрузку с серверов прокси по протоколу HTTP.

wget поддерживает очень много параметров, которыми можно управлять. Чтобы познакомиться с утилитой более детально используйте справочную информацию в вашей системе.

Загрузка содержимого сайта

wget -r позволяет рекурсивно загрузить содержимое указанного сайта.

Синтаксис: [Команда] [Параметр] [Ссылка на файл]

# Система загрузит содержимое сайта example.ru

$ wget -r www.example.ru

Загрузка файла с сайта

wget -c позволяет загрузить любой файл с сайта, с возможностью остановки и продолжения загрузки.

Синтаксис: [Команда] [Параметр] [Ссылка на файл]

# Система будет загружать файл test.iso с сайта www.example.ru

$ wget -c www.example.ru/test.iso

Загрузка файла с сайта в указанное время

Для загрузки файлов с сайта в нужное для вас время воспользуйтесь следующей командой.

Синтаксис: [Команда] [Параметр] [Ссылка на файл]

# Система начнёт загрузку файла test.iso с сайта www.example.ru в 09:00

$ echo 'wget -c www.example.ru/test.iso' | at 09:00
