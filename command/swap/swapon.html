swapon

На главную

Описание

Команда swapon позволяет активировать SWAP-пространство (а).

Синтаксис: [Команда] [Имя раздела]

Активация SWAP-пространства

Для того, чтобы активировать SWAP-пространство, воспользуйтесь следующей командой:

# Будет выполнена активация swap-пространства, расположенного на разделе hda3

$ swapon /dev/hda3

Активация SWAP-пространств

Для активации нескольких SWAP-пространств воспользуйтесь следующей командой:

# Будут активированы два swap-пространства, расположенные на разделах hda2 и hdb3

$ swapon /dev/hda2 /dev/hdb3
