Кнопка назад

shutdown

На главную

Описание

Команда shutdown прекращает работу всех процессов.

Синтаксис: [Команда] [Параметры] [Время] [Сообщение]

# Компьютер будет выключен немедленно

$ shutdown -h 0

# Компьютер будет выключен через 5 минут

$ shutdown -h +5

# Компьютер будет выключен через 5 минут с информационным сообщением

$ shutdown -h +5 "Компьютер будет выключен через 5 минут"

Параметры

-h - полная остановка системы (компьютер будет выключен).

-r - перезагрузить систему.
