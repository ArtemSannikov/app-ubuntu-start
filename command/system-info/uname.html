Кнопка назад

uname

На главную

Описание

Команда uname позволяет вывести информацию о системе.

Синтаксис: [Команда] [Параметр]

# Будет выведена версия ядра Linux

$ uname -a

# Будет отображена архитектура компьютера

$ uname -m

Параметры

-a - вывести версию ядра Linux.

-m - отобразить архитектуру компьютера.

-i - вывести имя платформы.

-r - вывести информации о релизе операционной системы.

-s - вывести имя операционной системы.

-v - вывести информацию о версии операционной системы.
