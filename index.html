rmdir [dir]

Удаление пустой директории

Команда (rmdir) позволяет удалить пустую директорию в терминале операционной системы Ubuntu.

//команда [имя папки]
$ rmdir test

rm -rf [dir]

Удаление директории с содержимым

Команда (rm -rf) позволяет удалить директорию вместе с содержимым в терминале операционной системы Ubuntu.

//команда [имя папки]
$ rm -rf dir1

Удаление двух директорий с содержимым

Команда (rm -rf) позволяет удалить сразу две директории с их содержимым в терминале операционной системы Ubuntu.

//команда [имя папки 1] [имя папки 2]
$ rm -rf dir1 dir2

rm -f [file]

Удаление файла

Команда (rm -rf) позволяет удалить любой файл в терминале операционной системы Ubuntu.

//команда [имя файла]
$ rm -f test.txt

pwd

Путь до текущей директории

Команда (pwd) показывает путь до текущей директории в терминале операционной системы Ubuntu.

$ pwd

mv [dir1] [dir2]

Переименовать директорию

Команда (mv) позволяет осуществить переименование директории в терминале операционной системы Ubuntu.

//команда [старое имя папки] [новое имя папки]
$ mv test dir1

Перемещение директории

Команда (mv) позволяет переместить директорию в терминале операционной системы Ubuntu.

//команда [имя папки] [путь до директории, куда нужно перенести папку]
$ mv test dir1

cp [file] [dir]

Копирование файла из директории в директорию

Команда (cp) позволяет осуществить копирование файла из одной директории в другую в терминале операционной системы Ubuntu.

//команда [имя файла] [путь до новой директории]
$ cp test.txt dir1

mkdir [dir1]

Создание директории

Команда (mkdir) позволяет создать новую директорию в терминале операционной системы Ubuntu.

$ mkdir [dir1]

Создание одновременно двух директорий

Команда (mkdir) позволяет создать одновременно две директории в терминале операционной системы Ubuntu.

$ mkdir [dir1] [dir2]

ls

Отображение содержимого директории

Команда (ls) позволяет отобразить содержимое директории в терминале операционной системы Ubuntu.

$ ls

ls -l

Детализированное отображение файлов и директорий

Команда (ls -l) выводит на экран детализированное отображение файлов и директорий в терминале операционной системы Ubuntu.

$ ls -l

ls -F

Отображение содержимого директории с характеристикой файлов

Команда (ls -F) выводит на экран содержимое директории с характеристикой файлов в термина операционной системы Ubuntu.

$ ls -F

ls -a

Показать скрытые файлы и директории

Команда (ls -a) показывает скрытые файлы и директории в терминале операционной системы Ubuntu.

$ ls -a

cd

Переход в любую директорию

Команда (cd) позволяет открыть любую директорию в терминале операционной системы Ubuntu.

$ cd [directory]
$ cd /home/artem/Видео/Вконтакте

cd ..

Переход в директорию уровнем выше

Команда (cd ..) позволяет перейти в директорию уровнем выше в терминале операционной системы Ubuntu.

$ cd ..

cd ../..

Переход в директорию двумя уровнями выше

Команда (cd ../..) позволяет перейти в директорию двумя уровнями выше в терминале операционной системы Ubuntu.

$ cd ../..

cd -

Переход в предыдущую директорию

Команда (cd -) позволяет перейти в предыдущую директорию в терминале операционной системы Ubuntu.

$ cd -

cd

Переход в домашнюю директорию

Команда (cd) позволяет быстро переместиться в домашнюю директорию в терминале операционной системы Ubuntu.

$ cd

whereis bash

Путь до командного интерпретатора bash

Команда (whereis bash) позволяет узнать путь до командного интерпретатора bash в терминале операционной системы Ubuntu.

$ whereis bash

!!

Выполнение предыдущей команды

Команда (!!) позволяет выполнить предыдущую команду в терминале.

//Последняя команда, которая была выполнена в терминале
$ cal -3
//команда, которая выполнит последнюю команду, то есть команду (cal -3)
$ !!

host

Как узнать IP адрес сайта

Команда (host) позволяет узнать IP-адрес любого сайта при помощи команды:

$ host [url web-site]
$ host artemsannikov.ru

hostname

Сетевое имя компьютера

Для того, чтобы вывести сетевое имя компьютера в терминале Ubuntu, воспользуйтесь следующей командой:

$ hostname

uptime

Время работы компьютера без перезагрузки

Для того, чтобы узнать время работы системы без перезагрузки и выключения в Ubuntu, воспользуйтесь следующей командой:

$ uptime

cal -3

Вывод предыдущего, текущего и следующего месяца

Для вывода предыдущего, текущего и следующего месяца в терминале Ubuntu, воспользуйтесь командой:

$ cal -3

uname -a

Текущая версия ядра Linux

Для того чтобы вывести версию ядра linux в операционной системе Ubuntu необходимо воспользоваться командой:

$ uname -a

clear

Очищение экрана терминала

Для того, чтобы очистить терминал от старых команд в Ubuntu, необходимо воспользоваться следующей командой:

$ clear

date

Вывод текущей даты и времени

Для вывода текущей даты и время в Ubuntu, воспользуйтесь командой:

$ date
