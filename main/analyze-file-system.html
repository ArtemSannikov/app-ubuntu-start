Анализ файловых систем

  • badblocks -v [dev]

    Проверить раздел на
    наличие bad-блоков.

  • fsck [dev]

    Проверить/восстановить целостность
    файловой системы.

  • fsck.ext2 [dev]

    Проверить/восстановить целостность
    файловой системы ext2.

  • e2fsck -j [dev]

    Проверить/восстановить целостность
    файловой системы ext3 с указанием,
    что журнал расположен там же.

  • fsck.ext3 [dev]

    Проверить/восстановить целостность
    файловой системы ext3.

  • fsck.vfat [dev]

    Проверить/восстановить целостность
    файловой системы fat.
