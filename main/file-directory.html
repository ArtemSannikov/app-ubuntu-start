Кнопка назад

Файлы и директории

  • cd

    Переход в домашнюю директорию

  • cd -

    Переход в предыдущую директорию

  • cd ../..

    Переход в директорию двумя уровнями выше

  • cd ..

    Переход в директорию уровнем выше

  • cd [directory]

    Переход в любую директорию

  • ls -a

    Отображение скрытых файлов и директорий

  • ls -F

    Отображение содержимого директории с характеристикой файлов

  • ls -l

    Детализированное отображение файлов и директорий

  • ls

    Отображение содержимого директории

  • mkdir [dir1]

    Создание директории

  • cp [file] [dir]

    Копирование файла из директории в директорию

  • mv [dir1] [dir2]

    Перемещение и переименование директории

  • pwd

    Путь до текущей директории

  • rm -f [file]

    Удаление файла

  • rm -rf [dir]

    Удаление директории(й) с содержимым

  • rmdir [dir]

    Удаление пустой директории
