Форматирование файловых систем

  • mkfs [dev]

    Создать linux-файловую систему
    на разделе [dev].

  • mke2fs [dev]

    Создать файловую систему
    ext2 на разделе [dev].

  • mke2fs -j [dev]

    Создать журналирующую файловую
    систему ext3 на разделе [dev].

  • mkfs -t vfat 32 -F [dev]

    Создать файловую систему
    FAT32 на разделе [dev].

  • fdformat -n [dev]

    Форматирование floppy-диска
    без проверки.
