Клавиатурные сочетания

Комбинации клавиш - это сочетания клавиш клавиатуры, с помощью которых произвлдятся такие действия, как открытие диалога настроек или быстрый доступ к определённому функционалу Терминала.

Файл

Shift + Ctrl + T - новая вкладка.

Shift + Ctrl + N - новое окно.

Shift + Ctrl + W - закрыть вкладку.

Shift + Ctrl + Q - закрыть окно.

Правка

Shift + Ctrl + C - копировать.

Shift + Ctrl + V - вставить.

Вид

F11 - на полный экран.

Ctrl + +- увеличить.

Ctrl + -- уменьшить.

Ctrl + 0- стандартный размер

Работа с вкладками

Ctrl + Page Up - переключить на предыдущую вкладку.

Ctrl + Page Down - переключиться на следущую вкладку.

Shift + Ctrl + Page Up - переместить вкладку влево.

Shift + Ctrl + Page Down - переместить вкладку вправо.

Alt + 1 - переключиться на вкладку 1.

Alt + 2 - переключиться на вкладку 2.

Alt + 3 - переключиться на вкладку 3.

Alt + 4 - переключиться на вкладку 4.

Alt + 5 - переключиться на вкладку 5.

Alt + 6 - переключиться на вкладку 6.

Alt + 7 - переключиться на вкладку 7.

Alt + 8 - переключиться на вкладку 8.

Alt + 9 - переключиться на вкладку 9.

Alt + 0 - переключиться на вкладку 0.

Другое

Shift + Ctrl + Up - прокрутить на одну строку вверх.

Shift + Ctrl + Down - прокрутить на одну строку вниз.

Работа с Bash

Ctrl + W - удалить слово.

Ctrl + U - удалить строку.

Ctrl + A - переместиться в начало строки.

Ctrl + E - переместиться в конец строки.

Alt + B - переместиться на одно слово назад.

Ctrl + F - переместиться на один символ вперёд.

Alt + F - переместиться на одно слово вперёд.

Ctrl + U - удалить строку целиком.

Alt + Backspace - удалить слово перед текущей позицией.

Ctrl + R - обратный инкрементный поиск по буферу истории.

Alt + P - обратный неинкрементный поиск по буферу истории.
