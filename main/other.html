Прочие команды

  • clear

    Очистка экрана терминала.

  • apropos [keyword]

    Выводит список команд, которые имеют
    отношение к ключевому слову.

  • man [program]

    Вызов руководства по работе
    с программой.

  • whatis [program]

    Отображает краткое описание указанной
    программы.

  • gpg -c [file]

    Зашифровать файл с помощью
    GNU Privacy Guard.

  • gpg [file.gpg]

    Дешифровать файл с помощью
    GNU Privacy Guard.

  • wget -r [example.ru]

    Загрузить рекурсивно содержимое
    сайта.

  • wget -c [example.ru]

    Загрузить файл с возможностью
    остановки и продолжения.

  • echo 'wget -c [example.ru]' | at 09:00

    Начать загрузку файла с сайта
    в указанное время.

  • ldd /usr/bin/ssh

    Выводит список библиотек, необходимых
    для работы ssh.

  • alias [alias name]='[command]'

    Назначить алиас (псевдоним)
    конкретной программе.
