Просмотр содержимого файлов

  • cat [file]

    Вывод содержимого файла.

  • tac [file]

    Вывод содержимого файла в обратном
    порядке.

  • more [file]

    Постраничный вывод содержимого
    файла.

  • less [file]

    Постраничный вывод содержимого файла с
    возможностью пролистывания.

  • head -n [file]

    Вывод нескольких строк файла.

  • tail -n [file]

    Вывести последние строки файла.

  • tail -f /var/log/messages

    Выводить содержимое файла
    на устройство вывода.
